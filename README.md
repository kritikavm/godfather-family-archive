# The Godfather Family Archive

An interactive information design project exploring the Corleone family across all four generations of Mario Puzo's The Godfather trilogy (1972–1990).

## Live Site

**[→ View the archive](https://YOUR-USERNAME.github.io/godfather-family-archive/)**

## What's Inside

- **Family Dossier** — Searchable, sortable database of all 28 family members across 4 generations
- **Family Tree** — 2D constellation view + 3D time-depth perspective with D3.js
- **☠ Kill Web** — Force-directed graph of every confirmed death across the trilogy
- **🗺 World Map** — Migration routes from Corleone, Sicily to New York
- **Timeline** — Lifespans of all 28 members across 110 years of history (1860–2000)

## Features

- Single-file app — open directly in any browser, no install needed
- Click any family member to open their full profile
- Photo upload system — add actor portraits, they persist when you save
- Export / import data as CSV
- 💾 Save File — bakes all your changes into a new self-contained HTML file

## The Family

28 members across 4 generations:
- **Gen I · The Old Country** — The Andolinis and Sciortinos of Sicily (1858–1901)
- **Gen II · The Don's Generation** — Vito and Carmela Corleone
- **Gen III · The Sons** — Sonny, Michael, Fredo, Connie, Tom Hagen and more
- **Gen IV · The Heirs** — Vincent, Anthony, Mary, and the next generation

All relationships mapped from Vincent Mancini Corleone's perspective — the heir of Godfather Part III.

## Built With

- [D3.js](https://d3js.org/) — family tree, kill web, timeline, map
- Vanilla HTML / CSS / JavaScript — single self-contained file
- Playfair Display SC + Lora — typography
- No frameworks, no build tools, no dependencies beyond D3

## About

Academic information design project exploring genealogy, narrative, and data visualization through the lens of The Godfather trilogy, directed by Francis Ford Coppola, based on the novel by Mario Puzo.
